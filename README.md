# Cuaderno Future Engeneers
Este equipo está formado por Adrián Ordóñez Hernández, Lorenzo Manuel Latorre Martínez y Guillermo Cuesta Luque, somos estudiantes de 3º del instituto IES Az-Zait. En este cuaderno podremos ver toda la información que va asignada a nuestro robot, incluiremos electrónica, mecánica, programación, problemas detectados y muchos más apartados.
