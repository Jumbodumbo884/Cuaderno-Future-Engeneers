# Cuaderno Future Engeneers
Este equipo está formado por Adrián Ordóñez Hernández, Lorenzo Manuel Latorre Martínez y Guillermo Cuesta Luque, somos estudiantes de 3º del instituto IES Az-Zait. En este cuaderno podremos ver toda la información que va asignada a nuestro robot, en la cual incluiremos electrónica, mecánica, programación, problemas detectados y muchos más apartados.
Para nuestro apartado de electrónica podemos distinguir entre diferentes elementos como el sensor de ultrasonidos, motor, placa arduino, el escudo, su respectivo portabaterías con sus baterías y los cables usados para unificar todos nuestros elementos.
También contamos con elementos mecánicos como el chasis de PLA, ruedas, soportes para nuestrosultrasonidos nombrados anteriormente, 
un eje direccional y finalmente un soporte de PLA para sostener nuestra dirección.
