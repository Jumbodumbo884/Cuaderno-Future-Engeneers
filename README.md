# Cuaderno Future Engeneers
Este equipo está formado por Adrián Ordóñez Hernández, Lorenzo Manuel Latorre Martínez y Guillermo Cuesta Luque, somos estudiantes de 3º del instituto IES Az-Zait. En este cuaderno podremos ver toda la información que va asignada a nuestro robot, en la cual incluiremos electrónica, mecánica, programación, problemas detectados y muchos más apartados.

Para nuestro apartado de electrónica podemos distinguir entre diferentes elementos como el sensor de ultrasonidos, motor, placa arduino, el escudo, su respectivo portabaterías con sus baterías y los cables usados para unificar todos nuestros elementos. Gracias a estos elementos nos proporcionarán la ayuda necesaria para la cual podamos programarlos y los mismos nos ayuden con sus espectivas funciones y así proporcionar un gran y buen funcionamiento en nuestro respectivo robot.

También contamos con elementos mecánicos como el chasis de PLA, ruedas, soportes para nuestrosultrasonidos nombrados anteriormente, 
un eje direccional y finalmente un soporte de PLA para sostener nuestra dirección.

Un apartado que también es muy necesario es el de programación, en el cual incluiremos todos los programas y ensayos que hayamos realizado con nuestro robot, se podrán incluir también todos los programs tanto de pruebas y definitivos para poderlos comparar respectivamente y saber con seguridad como esta realizad y cual es el correcto para usar.
