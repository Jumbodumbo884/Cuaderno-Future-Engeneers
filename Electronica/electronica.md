# Electrónica
## Componentes electrónicos del robot:
- Sensores ultrasonidos HC-SR04 (x3)
- Motor (x1)
- Placa arduino (x1)
- Escudo L298N (x1)
- Portabaterias (x1)
- Baterías recargables de 3'7 V, modelo TR 18650 de 9900 mAH (x2)
- Cables
## Para que se usa cada componente del robot:
- Sensores ultrasonidos HC-SR04: Estos se usan para poder saber y medir a que distancia esta la pared y los respectivos opstáculos.
- Motor: Es el que le va a dar potencia a las ruedas para que las mismas se puedan mover y desplazarse.
- Placa arduino: Es el cerebro que controla el robot.
- Escudo L298N: El escudo sirve para poder controlar los motores del robot.
- Portabaterías: Este utensilio nos ayuda a poder poner las baterías en el mismo.
- Baterías recargables de 3'7 V, modelo TR 18650 de 9900 mAH: Estas baterías son las que darán energía al robot.
- Cables: Son los encargados de unificar todos los componentes de arduino etc.
